## Coregistration

### Coregistration pipeline
In xDEM, it is possible to create coregistration pipelines. This allows a series of coregistrations to be performed sequentially.
This feature is also available in QGIS. To ensure the best possible integration of xDEM into the QGIS environment, the algorithms are compatible with the model designer.

[Model designer, QGIS documentation](https://docs.qgis.org/3.44/en/docs/user_manual/processing/modeler.html).
